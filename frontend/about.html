Changelog — v2.0

New
Vision Transformer (ViT) Model — Replaced CNN backbone with full ViT architecture using patch embeddings and multi-head self-attention for superior deepfake detection accuracy.
New
Complete Multi-Page Dashboard — Analytics, History, AI Assistant, Reports, Learn, Settings, and About pages with persistent session management.
New
AI Explainability — Every prediction now includes a human-readable explanation and key visual indicators explaining the model's decision.
New
Built-in AI Assistant — FAQ-powered chatbot that answers questions about deepfakes, ViT, MTCNN, and the Celeb-DF dataset with typing animations.
New
4 Theme System — Dark, Light, AMOLED, and Cyberpunk themes with instant CSS variable switching, persisted across sessions.
Improved
Image Preprocessing — MTCNN face detection with 30px margin, Cropper.js integration for manual face selection before analysis.
Improved
Analytics Charts — Chart.js powered donut, line trend, and confidence distribution bar charts with real-time data from localStorage history.
Improved
Export Capabilities — Download scan history as CSV, individual detection reports as text files, and PDF export via browser print.
Fix
Session Persistence — Sessions survive page navigation using sessionStorage with automatic history restoration across all sub-pages.
