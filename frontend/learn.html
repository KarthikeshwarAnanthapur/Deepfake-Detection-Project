Deepfake History Timeline

2014
GANs Invented by Goodfellow et al.
Ian Goodfellow introduces Generative Adversarial Networks, laying the mathematical foundation for synthetic image generation.
2017
First "Deepfake" Videos Appear
Reddit user "deepfakes" posts face-swapped celebrity videos using encoder-decoder GAN architecture, coining the term "deepfake".
2018
FaceApp & DeepFaceLab Go Mainstream
User-friendly deepfake tools emerge, enabling non-experts to create convincing face swaps. Detection research accelerates.
2019
Celeb-DF Dataset Released
A high-quality deepfake benchmark dataset of celebrity videos is released, enabling standardized evaluation of detection models.
2020
FaceForensics++ & DFDC
Facebook's Deepfake Detection Challenge attracts 2,000+ teams. Vision Transformers are introduced for image recognition.
2021
ViT Applied to Deepfake Detection
Researchers begin applying Vision Transformers to deepfake detection, showing superior accuracy over CNN-based approaches.
2022–2024
Diffusion Models & Audio Deepfakes
DALL-E, Stable Diffusion, and voice cloning tools create new detection challenges. DeepGuard v2.0 uses ViT for state-of-the-art detection.
